Гайд по статусам оборудования

Все статусы

Каждое оборудование имеет один из шести статусов:

  1. Работает (на складе)
  2. Не работает
  3. Работает (у клиента)
  4. Передан в филиал
  5. Списан
  6. В ремонте
Изменение статуса оборудования

Первые два статуса пользователь может задавать сам при создании или редактировании карточки оборудования. Изменять статус оборудования в карточке оборудования потребуется, если оборудование перестало работать или было починено в отделе. В остальных случаях статус у оборудования меняется в зависимости от созданных документов с этим оборудованием. Ниже представлена таблица, показывающая как статус оборудования меняется при создании и редактировании документов различных видов.

Созданный документ Старый статус оборудования Новый статус оборудования
Создание акта приема-передачи с контрагентами (без даты возврата) Работает (на складе) Работает (у клиента)
Добавление даты возврата в акте приема-передачи с контрагентами Работает (у клиента) Работает (на складе)
Создание акта приема-передачи с филиалами (без даты возврата) Работает (на складе) Передан в филиал
Добавление даты возврата в акте приема-передачи с филиалами Передан в филиал Работает (на складе)
Создание акта МФУ Не работает Списан
Создание акта приема-передачи с ремонтниками Не работает В ремонте
Создание акта выполненных работ (без отметки о неисправности) В ремонте Работает (на складе)
Создание акта выполненных работ (с отметкой о неисправности) В ремонте Не работает

При удалении документа статус меняется обратно. Например, при удалении акта приема-передачи с контрагентами (без даты возврата) статус оборудования с "Работает (у клиента)" поменяется на "Работает (на складе)".

Редактирование статуса оборудования в его карточке блокируется, если последним документом оборудования был один из документов из списка ниже. В ином случае, возможность изменить статус есть.

  1. Акт приема-передачи с контрагентами (без даты возврата)
  2. Акт приема-передачи с филиалами (без даты возврата)
  3. Акт МФУ
  4. Акт приема-передачи с ремонтниками
